Cronosfera
Mobile Web Application
Le Mobile Web Application sono un modo per fornire contenuti e funzionalità a dispositivi mobili quali iPhone, iPad e tutti gli smartphone e tablet. Su queste dispositivi le Mobile Web App rappresentano un’alternativa alle App tradizionali, vale a dire quelle che vengono normalmente scaricate e installate dagli store ufficiali. Essendo, per loro natura, fruibili via web le Mobile Web App non necessitano di alcuna installazione e di nessun tempo di download, si navigano e di utlizzano con il normale browser del dispositivo mobile. Per iniziare a utilizzare la Mobile Web App della Cronosfera, dopo aver cliccato su AVANTI, apri le opzioni del browser (ad esempio i tre puntini verticali in Google Chrome) e clicca su Aggiungi a schermata Home. Dopo averlo fatto, chiudi il browser e apri la Cronosfera dal'icona che è stata aggiunta sullo sfondo.
